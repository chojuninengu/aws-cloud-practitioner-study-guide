## 🎯 Exam Information
- **Exam Code**: CLF-C02
- **Duration**: 90 minutes
- **Questions**: 65 questions (50 scored, 15 unscored)
- **Passing Score**: 700/1000
- **Cost**: $100 USD

## 📊 Exam Domains
1. **Cloud Concepts** (24%)
2. **Security & Compliance** (30%)
3. **Technology** (34%)
4. **Billing & Pricing** (12%)
